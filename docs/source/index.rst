Ansible Collection for OpenAFS
==============================

This is an Ansible collection to deploy the `OpenAFS`_ distributed filesystem.
The collection provides roles, modules, and example playbooks to install and
configure OpenAFS clients and servers.

.. _`OpenAFS`: https://www.openafs.org
